# 專題方向腦力激發

- 會議日期: 2025-09-25
- 會議類型: 決策會議
- 與會者: 邱偉豪, 庭語 張, 陳秉鴻, 張 家瑜, 鍾百陶

## 討論內容

核心要求：以 LLM 為核心、重視 API 成本與商業化潛力。

提案一（資訊過載）：整合 104、591 等資訊，風險在於功能重疊、反爬與法遵、競品強。

提案二（防詐騙）：文字偵測（Web/擴充）與語音偵測（App），需評估 STT 成本、蒐集詐騙資料，社會價值高、共識最高。

提案三（學貸與理財）：學貸規劃與綜合理財，特色為遊戲化、Web+App，但競品多、串金融 API 難度高。

## 決議事項

- 主要主題：防詐騙應用。
- 備案順序：第一備案—學貸規劃工具；第二備案—資訊整合／理財平台。
- 優先級：先做文字詐騙偵測（Web/擴充），再挑戰語音詐騙偵測（App）。
